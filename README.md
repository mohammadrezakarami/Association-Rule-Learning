# Association Rule Learning

This repository contains notebook implementations of association rule learning methods on market basket data.

## Repository Contents

- `apriori.ipynb`: Apriori algorithm workflow and analysis.
- `eclat.ipynb`: Eclat algorithm workflow and analysis.
- `Market_Basket_Optimisation.csv`: dataset used for rule mining experiments.

## Topics Covered

- market basket analysis
- Apriori
- Eclat
- frequent itemset mining
- association rule extraction

## Getting Started

1. Clone the repository.
2. Open the notebooks in Jupyter.
3. Make sure the required Python libraries for data analysis are installed.
4. Run the notebooks to reproduce the rule mining experiments.

## Notes

This repository is intended as an educational implementation of common association rule learning techniques.
